Usage

Run a command inside the container:

1
2
3
4

# Using docker:
docker run ghcr.io/serdigital64/<IMAGE> <COMMAND>
# Using podman:
podman run ghcr.io/serdigital64/<IMAGE> <COMMAND>

Deployment

Requirements

  • Container engine
    • docker or podman
  • Bash

Installation

Download the target image to the local registry:

2
3
4
